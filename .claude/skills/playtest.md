# Playtest

Launch the game preview server and take a screenshot to check the current state of the game visually.

## Steps

1. Start the preview server using `preview_start` with the "colony-wars" configuration
2. Take a screenshot with `preview_screenshot` to see the current state
3. Report what you see — layout, rendering issues, missing elements, etc.
